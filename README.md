# **[Hotel Browser]**

## Descripción

[Mi aplicación se basa en un explorador de hoteles. En ella debemos crear una cuenta para iniciar sesión como administrador, la idea principal solo contempla un usuo administrativo y no por un cliente directo. 

Una vez sesión iniciada podemos ver varios apartados en el menú lateral, en el primero un perfil del usuario con la sesión iniciada con la opción del cierre de sesión. Como segundo apartado encontramos los servicios que podremos crear o eliminar a nuestro gusto. En el tercer apartado vemos las reservas que se han realizado, a su vez podemos eliminarlas y realizar una reserva propia desde esta página.

El siguiente apartado es un registro de todos los usuarios registrados en la aplicación, en la que podemos eliminarlos excepto el actual usuario loggeado. Una página de mapa incluye la ubicación de cada servicio registrado en el apartado de servicios, utilizando la ubicación insertada para obtener las coordenadas y mostrarlas en el mapa interactivo.

Por úlimo tenemos el apartado de 'Sobre mí' que incluye información mia y un enlace a mi GitHub. A su vez encontramos un apartado para cambiar el idioma de la aplicación, ahora mismo solo están disponible el Español, Inglñes, Alemán e Indú pero más adelante podré añadir cualquier otro.

Es completamente responsive para dispositivos móviles de cualquier resolución.]
